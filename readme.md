# Modern forum by VuongNgo
Aim to modernize forum interface using Reactjs && Material Design.   
Proof of concept backed by Meteorjs.   
[Demo](http://188.166.213.121).   
 
### Current stacks
* Reactjs
* Meteorjs
* Webpack
* ES2015
* Material-ui
 
### Development mode
* With test:   
    meteor
* No test:   
    VELOCITY=0 meteor
     
### Production mode
    meteor --production

### Build
    meteor build .

### Deploy with MUP
    mup deploy
